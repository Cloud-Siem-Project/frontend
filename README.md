# AWSIEM Frontend

Bu repo, AWSIEM ürününün frontend bileşenlerini içerir. Güvenlik etkinliklerini görüntüleme, yönetme ve analiz etmeyi kolaylaştırmak için kullanıcı dostu, basit ve interaktif bir arayüz sunar.

# Tech Stack
React – UI library
React Router – Routing and navigation
JavaScript (ES6+)
CSS - Styling

# Project Structure

frontend/
│── src/
│   ├── components/      # Reusable UI components
│   ├── pages/           # Main pages (Dashboard, Nodes, Alerts, etc.)
│   ├── layout/          # Layout components (Sidebar, Navbar)
│   ├── routes/          # Route definitions
│   ├── services/        # API calls
│   ├── assets/          # Static files
│   └── App.jsx          # Root component
│
│── public/              # Static public assets
│── package.json         # Dependencies & scripts
│── README.md            # Project documentation
